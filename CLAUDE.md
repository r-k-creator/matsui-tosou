# 松井塗装 ホームページ運用ルール

## 基本方針
- このフォルダは松井塗装のホームページ（index.html）を管理しています
- 公開URL: https://r-k-creator.github.io/matsui-tosou/
- GitHubリポジトリ: https://github.com/r-k-creator/matsui-tosou
- Googleビジネスプロフィール: 登録済み
- Google Search Console: 登録・所有権確認済み
- Google Analytics（GA4）: 導入済み（測定ID: G-QZB72SE62W）

## 自動で進めてよいこと（確認不要）
- index.htmlの文言修正、CSSの調整
- 新しいセクションの追加（内容が明確に指示された場合）
- ファイルの内容確認・報告
- 運用メモ.mdへの追記

## 必ず一言確認してから進めること
- git commit・git push の実行（変更内容を要約して確認を求めること）
- 既存の重要な情報（電話番号・住所・料金・営業時間など）を変更する場合
- ファイルの削除

## 作業前に必ず確認すること
- 運用メモ.md に、過去の経緯・注意点・定型の作業手順が記録されています。作業前に必ず目を通してください
- 特に「施工事例の差し替え手順」「GBP投稿文」のセクションは、関連する依頼が来たときに参照してください

## 過去のトラブルと注意点
- Google Apps Script（GAS）のコードを修正した場合、保存だけでは公開中のURLに反映されません。必ず「デプロイ」→「新しいデプロイ」を実行し直してください
- GitHub Pagesは反映に数十秒〜数分かかることがあります。すぐに反映されていなくても、少し時間を置いて再確認してください
- ファイルを編集する際は、必ず現在のファイルの中身を確認してから修正してください（古いバージョンをベースに編集すると、既存の修正が消えてしまう「巻き戻り」が起きます）
